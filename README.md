# Hospital Emergency Room Analysis Dashboard

## Project Overview
This Power BI project analyzes hospital emergency room operations to understand patient flow,
wait times, satisfaction levels, and department referrals.

## Objectives
- Monitor number of patients visiting the ER
- Analyze average wait time and patient satisfaction
- Track admission status and department-wise referrals
- Support data-driven decision-making for hospital management

## Dashboards Included
1. Monthly View
2. Consolidated View
3. Patient Details
4. Key Takeaways

## Tools & Technologies
- Power BI
- DAX
- Data Modeling
- Data Visualization

## Key Insights
- Identified peak hours and high patient inflow periods
- Analyzed factors affecting patient wait time
- Highlighted departments with higher referral rates
